# Sprout Lands

A cozy farming simulation where you plant crops, manage your garden, and gather resources through day and night cycles.

## Controls

- **Movement**: WASD
- **Interact**: Space
- **Plant seeds**: Ctrl
- **Sleep/Shop**: Enter
- **Switch tools**: Q
- **Switch seeds**: E

## Installation

   ```bash
   pip3 install -r requirements.txt
   python3 src/main.py
   ```
